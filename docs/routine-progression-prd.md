# 전신/분할 루틴 진행 가이드 PRD

## 1. 제품 정의
Smart Trainner의 전신/분할 기능은 요일 고정 스케줄러가 아니라 `1일차~N일차`로 이어가는 루틴 진행 가이드다. 사용자는 앱을 열었을 때 오늘 어떤 부위를 집중할지 바로 알고, 운동을 완료하면 다음 일차로 자연스럽게 넘어간다.

핵심 포지셔닝:

> Smart Trainner는 초보자~중급자가 전신 또는 분할 루틴을 선택해, 요일에 묶이지 않고 오늘 집중할 운동을 이어가도록 돕는 루틴 진행 가이드다.

## 2. 문제 정의
초보자~중급자는 루틴 선택과 지속 진행에서 다음 문제를 겪는다.

- 오늘 어떤 운동을 해야 하는지 매번 고민한다.
- 전신, 분할, 부위 집중 분할의 차이를 명확히 이해하기 어렵다.
- 요일 기반 루틴은 하루 놓치면 전체 계획이 꼬였다고 느낀다.
- 분할 루틴을 원하지만 상체/하체만으로는 등, 가슴, 하체, 어깨, 팔을 집중하는 맛이 부족하다.
- 반대로 부위 집중 분할을 자유롭게 열면 특정 부위 편식, 낮은 빈도, 회복 실패가 생길 수 있다.

## 3. 대상 사용자
### 포함
- 초보자: 최근 3~6개월 이상 일관된 근력운동 기록이 없거나 기본 동작, RPE, 점진 과부하에 익숙하지 않은 사용자.
- 중급자: 최소 3~6개월 이상 운동 기록이 있고 세트, 반복, 중량, 피로를 어느 정도 조절할 수 있는 사용자.
- 헬스장 중심 사용자.
- 주 2~5회 운동 가능 사용자.

### 제외
- 재활, 통증 치료, 의료 목적 사용자.
- 선수, 대회 준비, 고급 보디빌딩/파워리프팅 사용자.
- 체형 분석, 사진/영상 기반 자세 진단을 기대하는 사용자.

## 4. 핵심 원칙
- 요일을 정하지 않는다. 루틴은 `1일차 → 2일차 → 3일차 → 반복`으로 진행한다.
- 홈 화면의 중심은 `오늘은 등 집중`처럼 오늘 할 운동의 정체성이다.
- `전신/분할`은 전문 용어로 던지지 않고 사용자가 느끼는 운동 감각으로 설명한다.
- 사용자는 집중하고 싶고, 앱은 루틴이 망가지지 않게 잡아줘야 한다.
- MVP는 AI 처방 엔진이 아니라 검증된 프리셋 기반 루틴 진행 도구다.

## 5. 루틴 구조
### 전신
매번 온몸을 골고루 다루는 구조.

예:
- 1일차 전신 A
- 2일차 전신 B
- 3일차 전신 C

추천:
- 초보자 주 2~3회
- 운동 습관 형성
- 쉬는 날이 불규칙한 사용자

### 균형 분할
빈도와 회복 균형을 우선하는 분할 구조.

예:
- 상체 / 하체
- Push / Pull / Legs
- 상체 / 하체 / 전신

추천:
- 초보자 후반~중급자
- 주 3~4회
- 전체 균형을 유지하면서 분할을 시작하려는 사용자

### 부위 집중 분할
하루에 한 부위를 명확히 집중하는 헬스장식 분할 구조.

예:
- 등 집중
- 가슴 집중
- 하체 집중
- 어깨+팔 집중
- 유산소/코어 또는 컨디셔닝

추천:
- 운동 경험이 있는 사용자
- 주 4회 이상 가능 사용자
- 오늘 어떤 부위를 하는지 분명한 루틴을 원하는 사용자

## 6. MVP 사용자 흐름
1. 사용자가 온보딩 또는 플랜 화면에서 루틴 찾기를 시작한다.
2. 앱은 다음 질문을 묻는다.
   - 일주일에 몇 번 운동할 수 있나요? `2회 / 3회 / 4회 / 5회`
   - 한 번에 어느 정도 운동할 수 있나요? `30분 / 45분 / 60분`
   - 운동 경험은 어느 정도인가요? `거의 처음 / 조금 해봄 / 꾸준히 해봄`
   - 어떤 운동 느낌을 원하나요? `매번 전신을 골고루 / 오늘의 부위를 집중 / 앱이 추천해줘`
3. 앱은 추천 루틴 1개와 대안 1~2개를 보여준다.
4. 사용자가 루틴을 선택한다.
5. 홈 화면은 다음 운동을 `오늘은 등 집중`, `2일차 · 당기는 운동 · 약 50분`처럼 보여준다.
6. 사용자가 운동을 완료하면 다음 일차로 이동한다.
7. 마지막 일차를 완료하면 다시 1일차로 순환한다.

## 7. 추천 매트릭스
| 사용자/횟수 | 추천 |
|---|---|
| 초보자 주 2회 | 전신 A/B |
| 초보자 주 3회 | 전신 A/B/C |
| 초보자 주 4회 | 전신 3회 + 가벼운 보조 1회 또는 쉬운 부위 집중 4일 |
| 중급자 주 3회 | 전신 A/B/C 또는 Push/Pull/Legs 변형 |
| 중급자 주 4회 | 부위 집중 4일: 등/가슴/하체/어깨+팔 |
| 중급자 주 5회 | 부위 집중 5일: 가슴/등/하체/어깨/팔+유산소 |

## 8. 부위 집중 정책
- MVP는 자유 조립을 제공하지 않는다.
- 부위 집중은 앱이 검증한 프리셋으로만 제공한다.
- 사용자가 가슴, 팔 같은 특정 부위만 반복하는 구조는 허용하지 않는다.
- 각 Day는 집중 부위 60~75%, 보조 부위 25~40%로 구성한다.
- 어깨 Day는 전면보다 측면/후면 중심을 우선한다.
- 팔 단독 Day는 중급자 5일 루틴에서만 제한적으로 허용한다.
- 유산소는 근력 부위를 대체하는 Day가 아니라 컨디셔닝/회복 성격으로 정의한다.
- 각 Day에는 최소 휴식 권장 시간이 있다.

## 9. 홈 화면 UX
홈 화면은 운동 목록이 아니라 다음 운동 정체성을 먼저 보여준다.

예:

```text
오늘은 등 집중
2일차 · 당기는 운동 · 약 50분

주요 부위
광배 · 등 중앙 · 후면 어깨 · 이두

오늘 운동 예시
랫풀다운 · 시티드 로우 · 페이스풀

[운동 시작]
다음 운동: 가슴 집중
```

## 10. 루틴 카드 UX
루틴 카드는 전문 용어보다 사용자의 기대와 결과를 먼저 보여준다.

예:

```text
부위 집중 4일 루틴
등 · 가슴 · 하체 · 어깨+팔을 하루씩 집중해요

추천 대상: 운동을 조금 해본 사용자
횟수: 주 4회 권장
시간: 45~60분
진행: 1일차 → 2일차 → 3일차 → 4일차 반복
```

## 11. MVP 포함 범위
- 온보딩 질문: 횟수, 시간, 경험, 원하는 운동 느낌.
- 추천 루틴 1개 + 대안 1~2개.
- 전신 루틴.
- 균형 분할 루틴.
- 부위 집중 분할 루틴.
- 1일차~N일차 순환 진행.
- 홈 화면의 오늘 집중 부위 표시.
- 완료 시 다음 일차 이동.
- 쉬었다가 와도 다음 일차에서 이어가기.
- 루틴 카드에 추천 대상, 시간, 횟수, 집중 부위 표시.

## 12. MVP 제외 범위
- 요일 고정 캘린더 스케줄러.
- 완전 자유형 루틴 생성.
- AI 맞춤 보디빌딩 처방.
- 부상/재활 판단.
- 약점 부위 분석.
- 체형 기반 추천.
- 고급 주기화.
- 자동 디로드.
- 영상 기반 자세 진단.

## 13. 수용 기준
- 사용자는 요일을 정하지 않아도 루틴을 시작할 수 있다.
- 앱은 다음 운동을 `N일차 + 집중 부위`로 보여준다.
- 분할 루틴에는 부위 집중형 프리셋이 포함된다.
- 초보자에게 고강도 5분할을 기본 추천하지 않는다.
- 부위 집중 루틴은 전체 순환이 깨지지 않게 구성된다.
- 전신/균형 분할/부위 집중의 차이를 카드 설명만으로 이해할 수 있다.
- 사용자는 운동을 완료하면 다음 일차로 자연스럽게 넘어간다.
- 마지막 일차 완료 후 1일차로 순환한다.
- 사용자에게 보이는 문구는 `values`와 `values-ko` 리소스로 관리한다.

## 14. 성공 지표
- 온보딩 후 추천 루틴 시작률.
- 첫 세션 완료율.
- 7일 또는 14일 내 두 번째 세션 완료율.
- 사용자가 다음 운동을 이해했다고 응답하는 비율.
- 루틴 중도 포기 사유 중 “너무 어렵다/너무 쉽다/장비가 없다” 비율.
- 완료 후 다음 일차로 자연스럽게 이어지는 비율.
