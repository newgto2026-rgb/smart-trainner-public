## Fixed Review Order

For each exercise:

1. Confirm catalog entry and mapped step images.
2. Confirm every shipped step image is exactly `720 x 800 px`.
3. Confirm thumbnail is exactly `360 x 400 px` and derived from an accepted step image.
4. Check background: single warm cream/off-white canvas, no gym room, floor seam, wall boundary, square panel patch, or source-sheet edge.
5. Check character: one allowed Smart Trainer coach variant per exercise sequence; do not mix variants inside one exercise.
6. Check crop/scale: no clipped head, hands, feet, load path, bar ends, handles, weight stack, pedals, bench, box, or machine base.
7. Check form: image sequence must match the written exercise and not teach unsafe or different movement mechanics.
8. Mark status as `Pass`, `Needs regeneration`, `Needs crop/export fix`, `Needs copy fix`, or `Representative only`. Use `Pass` for exported images that have been inserted into app resources; if a later visual defect is confirmed, downgrade only that row to the specific fix status.
